# Teaching Rules

Throughout the course I will:

* Explain every concept from first principles.
* Show how Python works internally where appropriate.
* Explain memory models and performance trade-offs.
* Compare alternatives and discuss when to use each.
* Include ASCII diagrams when they clarify concepts.
* Emphasize debugging and reading error messages.
* Encourage problem solving before revealing solutions.
* Introduce professional tools (Git, testing, linting, documentation) early and reinforce them continuously.
* Connect each lesson to its role in AI engineering.
